# Digital Forensics Case Study

## Objective
Analyze disk and memory artifacts to identify suspicious activity and create a forensic timeline.

## Tools Used
- Autopsy
- Volatility 3
- Plaso
- FTK Imager

## Key Findings
- USB-based data exfiltration detected
- Suspicious process activity in memory
- File deletion attempts observed

## Skills Demonstrated
- Disk forensics
- Memory analysis
- Timeline creation
- Evidence documentation
